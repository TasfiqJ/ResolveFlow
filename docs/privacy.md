# Privacy note

The public snapshot product uses fictional synthetic incident data and no analytics, advertising, cookies, accounts, uploads, or free-form input. It makes no request to Slack or Jira.

The optional local review workflow does not transmit responses. Review exports, if genuinely collected later, must use pseudonymous reviewer IDs, aggregate roles/familiarity, avoid employer-confidential information, and remain private until consent and publication review are recorded.

The optional local live API hashes the presented network address for quota enforcement; it does not expose the hash publicly. Retention and rotating-salt policy must be set before any nonlocal live mode is authorized.
